### 后端架构模式

- **三层架构**：Controller → Service → Mapper
- **DTO 模式**：请求/响应对象与数据库实体分离
- **RESTful API**：统一 `Result<T>` 响应格式
- **全局异常处理**：`@RestControllerAdvice`
- **拦截器模式**：JWT 认证拦截 + 接口限流拦截
- **无状态认证**：JWT Bearer Token

### 前端架构模式

- **SPA（单页应用）**：Vue Router hash 模式
- **组合式 API**：`<script setup>` + `ref`/`reactive`
- **拦截器模式**：Axios 请求自动带 Token + 响应统一错误处理
- **SSE 流式接收**：`fetch` + `ReadableStream` 逐行解析
- **懒加载**：`() => import(...)` 按需加载页面组件

## 运维 / DevOps

| 技术 | 版本 | 作用 |
|------|------|------|
| Nginx | 1.26.3 | 反向代理 + 静态文件服务 + HTTPS |
| Let's Encrypt | — | 免费 SSL 证书（HTTPS） |
| Systemd | — | 后端开机自启服务 |

### 部署架构

- **开发环境**：前后端分离（Vite :5173 + SpringBoot :8080，CORS 跨域）
- **生产环境**：Nginx 统一入口（:80/:443），消除跨域
- **HTTPS**：HTTP → 301 跳转 HTTPS，HSTS 强制加密
- **静态缓存**：带 hash 的资源文件 1 年缓存，index.html 不缓存

## 安全

| 机制 | 技术 | 说明 |
|------|------|------|
| 密码加密 | BCrypt | 不可逆 + 自动加盐 |
| 身份认证 | JWT (HMAC-SHA384) | 无状态，7 天过期 |
| 密码强度 | 自定义 `@StrongPassword` | ≥8位 + 大小写 + 数字 |
| 登录防暴破 | IP 限流 5次/分 | `RateLimitInterceptor` |
| 聊天防刷 | 用户 10次/分 + IP 20次/分 | 同上 |
| 会话越权防护 | `validateOwnership()` | 校验 session 归属 |
| 异常信息保护 | `BusinessException` | 内部细节不泄露 |
| 密码哈希保护 | `@JsonIgnore` | 序列化时跳过 password |
| 参数校验 | `@Valid` + Bean Validation | DTO 层校验 |
| 全局异常处理 | `@RestControllerAdvice` | 8 种异常类型覆盖 |
| CORS | `CorsConfig`（仅开发环境） | 生产由 Nginx 同源消除 |
| XSS 防护 | CSP + X-XSS-Protection | Nginx 安全头 |
| HTTPS | HSTS + TLS 1.2/1.3 | 传输加密 |
